# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/LeonneBrito/dash.go.git
```

```bash
# Install Dependencies
$ yarn
```

```bash
$ yarn dev
```

Go to http://localhost:3000/ to see the result.
